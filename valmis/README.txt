Randomizenator

Read Me in English bellow.

Randomizenator on helppo työkalu kaiken tasoisille käyttäjille.

Ohjelma käynnistyy ”Generate”-välilehteen johon käyttäjä voi 
lisätä haluamansa käyttäjänimen tai muuten salasanaa kuvaavan
aputunnuksen, käyttäjänimen voi myös jättää tyhjäksi.
Jos haluat vain nopean oletus salasanan
(8 merkkiä ja vain pieniä kirjaimia), paina ainoastaan
”Generate”-nappia ohjelman käynnistyttyä ja olet valmis.
Halutessasi voit tallentaa käyttäjänimi-salasana parin
tekstitiedostoon(.txt) painamalla ”Save As”-painiketta.
Ohjelma osaa liittää tiedoston perään itse tiedostomuodon,
joten käyttäjä kirjoittaa vain valitun tiedoston nimen.
Mikäli käyttäjä valitsee jo olemassa olevan tiedoston,
tiedot tallennetaan tiedoston perään.
Tiedostot tallennetaan muotoon:
'yourusername : yourpassword'

”Options”-välilehdeltä löydät salasanaan vaikuttavat
parametrit, kuten merkkien määrän tai isot kirjaimet. 

”Help”-välilehti sisältää samat tiedot kuin tämä,
mutta lyhemmin.

IN ENGLISH

Randomizenator is an easy and fast tool for beginners
and veterans alike.

The software starts up to "Generate"-tab where user can
give his/her username or other tag. Username can be also
left empty. If the user wants only quick default
password (8 characters and only small letters), user can
hit "Generate"button as the Randomizenator starts and 
it's done. User can save his/her username-password by 
pressing "Save AS" button. Software will save your file
as .txt by default. If selected file is already existing,
the data is appended after last line. The data is saved as:
'yourusername : yourpassword'

"Options" tab can be used to give different parameters to
create passwords with.

"Help" tab is short version of README.
